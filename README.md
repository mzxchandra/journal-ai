# journal-ai
Web App helping elderly users journal. Uses API calls to Google's Gemma model to generate prompts and ask follow-up questions.

User Research, Wireframes, Requirements, and more:

Backend: Flask

Frontend: HTML

Other Tools: Huggingface API
